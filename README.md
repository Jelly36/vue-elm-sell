# vue-elm-mall

## 线上预览

[https://konglingwen94.github.io/vue-elm-sell/dist](https://konglingwen94.github.io/vue-elm-sell/dist)

## 安装依赖

```
npm install
```

### 开启一个具有热重载功能的开发服务器

```
npm run serve
```

### 构建项目

```
npm run build
```
